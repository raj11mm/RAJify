## 🚀 Overview

**RAJify** is a full-stack e-commerce web application built using **Spring Boot, MySQL, HTML, CSS, and JavaScript**. It demonstrates complete end-to-end development including authentication, cart management, REST API integration, and a responsive UI.

⚡ Built rapidly with AI assistance while focusing on **understanding, debugging, and clean implementation**.

## ⚙️ Setup & Installation

### 1. Clone the repository

```
git clone https://github.com/raj11mm/RAJify.git
```

### 2. Navigate to project

```
cd RAJify
```

### 3. Configure MySQL

Create database and update `application.properties`:

```
spring.datasource.url=jdbc:mysql://localhost:3306/ecommerce_db
spring.datasource.username=YOUR_USERNAME
spring.datasource.password=YOUR_PASSWORD
```

### 4. Run application

```
mvn spring-boot:run
```

### 5. Open in browser

```
http://localhost:8080
```

## 🔗 API Endpoints (Sample)

* `GET /products` → Fetch products
* `POST /auth/signup` → Register user
* `POST /auth/login` → Login user
* `POST /cart/add` → Add to cart
* `GET /cart` → Get cart items
